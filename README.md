# تطبيق ترتيل - تطبيق القرآن الكريم التفاعلي

## 📱 وصف التطبيق
تطبيق ترتيل هو تطبيق Flutter متطور لقراءة وتلاوة القرآن الكريم مع ميزات ذكية لتحسين التلاوة باستخدام الذكاء الاصطناعي.

## ✨ الميزات الرئيسية

### 📖 التلاوة الذكية
- تحسين التلاوة بالذكاء الاصطناعي
- تحليل دقة النطق
- تقييم فوري للتلاوة
- تسجيل التلاوة ومقارنتها بالصوت الأصلي

### 📚 التجويد التفاعلي
- تعلم قواعد التجويد الأساسية
- شرح تفصيلي للقواعد مع أمثلة
- تمارين تفاعلية لكل قاعدة
- تدريب عملي على النطق الصحيح
- قواعد التجويد المتقدمة:
  - الغنة
  - المد
  - الإدغام
  - القَلْقَلَة
  - الإخفاء

### 💡 التفسير المتقدم
- تفسير ميسر للآيات
- شرح بسيط وواضح
- مراجع موثوقة من أشهر كتب التفسير:
  - التفسير الميسر
  - تفسير ابن كثير
  - تفسير الطبري
  - تفسير القرطبي
- عرض تفصيلي للآيات مع التفسير

### 🎯 الحفظ التفاعلي
- برنامج الحفظ التفاعلي
- تتبع التقدم في الحفظ
- اختبارات دورية للحفظ
- إحصائيات مفصلة للتقدم
- نظام المكافآت والإنجازات
- مراجعة دورية للمحفوظ

### 🔍 البحث الذكي
- بحث متقدم في القرآن الكريم
- البحث في السور والآيات والكلمات
- فلاتر متعددة للبحث
- اقتراحات سريعة للبحث
- نتائج بحث منظمة ومفصلة

### 📊 الإحصائيات والتقدم
- إحصائيات شاملة للنشاط
- رسوم بيانية للتقدم
- تتبع الوقت المستغرق
- عدد الآيات المقروءة
- السور المكتملة
- نقاط التقدم والإنجازات
- تفاصيل النشاط اليومي

### 🎵 التلاوات الصوتية
- مجموعة متنوعة من القراء
- جودة صوت عالية
- تحكم كامل في التشغيل
- إعدادات صوت متقدمة
- قائمة المفضلة

## 🛠️ التقنيات المستخدمة

### Frontend
- **Flutter**: إطار عمل متعدد المنصات
- **Dart**: لغة البرمجة
- **Provider**: إدارة الحالة
- **Material Design**: واجهة المستخدم

### Backend & APIs
- **HTTP**: للاتصال بالخوادم
- **alquran.cloud API**: بيانات القرآن الكريم
- **SharedPreferences**: حفظ البيانات المحلية
- **Audio**: تشغيل الصوت

### الميزات التقنية
- **Caching**: تخزين مؤقت للبيانات
- **Parallel Processing**: معالجة متوازية للبيانات
- **RTL Support**: دعم اللغة العربية
- **Responsive Design**: تصميم متجاوب
- **Performance Optimization**: تحسين الأداء

## 🚀 التثبيت والتشغيل

### المتطلبات
- Flutter SDK (3.0 أو أحدث)
- Dart SDK
- Android Studio / VS Code
- جهاز Android أو iOS أو محاكي

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone https://github.com/WaleedAbozeid/tarteel.git
cd tarteel
```

2. **تثبيت التبعيات**
```bash
flutter pub get
```

3. **تشغيل التطبيق**
```bash
flutter run
```

### بناء التطبيق للإنتاج

**لنظام Android:**
```bash
flutter build apk --release
```

**لنظام iOS:**
```bash
flutter build ios --release
```

## 📁 هيكل المشروع

```
lib/
├── main.dart                 # نقطة البداية
├── models/                   # نماذج البيانات
│   ├── quran_models.dart     # نماذج القرآن
│   └── audio_models.dart     # نماذج الصوت
├── providers/                # مزودي الحالة
│   ├── quran_provider.dart   # مزود بيانات القرآن
│   └── audio_provider.dart   # مزود الصوت
├── services/                 # الخدمات
│   ├── quran_service.dart    # خدمة القرآن
│   ├── audio_service.dart    # خدمة الصوت
│   └── local_data_service.dart # خدمة البيانات المحلية
├── screens/                  # الشاشات
│   ├── home_screen.dart      # الشاشة الرئيسية
│   ├── surah_screen.dart     # شاشة السورة
│   ├── tajweed_screen.dart   # شاشة التجويد
│   ├── tafsir_screen.dart    # شاشة التفسير
│   ├── memorization_screen.dart # شاشة الحفظ
│   ├── search_screen.dart    # شاشة البحث
│   ├── statistics_screen.dart # شاشة الإحصائيات
│   ├── recitation_screen.dart # شاشة التلاوة
│   ├── reciter_selection_screen.dart # اختيار القارئ
│   └── audio_settings_screen.dart # إعدادات الصوت
├── widgets/                  # العناصر المخصصة
│   ├── surah_card.dart       # بطاقة السورة
│   ├── feature_card.dart     # بطاقة الميزة
│   └── ayah_widget.dart      # عنصر الآية
└── utils/                    # الأدوات المساعدة
    ├── constants.dart        # الثوابت
    └── helpers.dart          # الدوال المساعدة
```

## 🔧 التحسينات المطبقة

### تحسينات الأداء
- **Caching System**: نظام تخزين مؤقت للبيانات
- **Parallel Processing**: معالجة متوازية للترجمات
- **Lazy Loading**: تحميل تدريجي للبيانات
- **Memory Management**: إدارة ذكية للذاكرة

### تحسينات واجهة المستخدم
- **RTL Support**: دعم كامل للغة العربية
- **Responsive Design**: تصميم متجاوب
- **Dark Theme**: مظهر داكن مريح للعين
- **Smooth Animations**: حركات سلسة

### إصلاحات الأخطاء
- **setState() Error Fix**: إصلاح أخطاء تحديث الحالة
- **Build Phase Issues**: حل مشاكل مرحلة البناء
- **Memory Leaks**: منع تسرب الذاكرة
- **Performance Bottlenecks**: إزالة نقاط الضعف
